# AI-Powered Workplace Safety and Compliance Monitoring

This project aims to optimize workplace compliance and safety specifically tailored for **pharmaceutical laboratories**. By utilizing Computer Vision and Deep Learning techniques, the system actively monitors lab environments to ensure strict adherence to safety protocols, detect missing Personal Protective Equipment (PPE), and identify unsafe behaviors to prevent accidents and contamination.

## Features

- **PPE Detection:** Real-time object detection of essential safety gear required in pharmaceutical settings (e.g., lab coats, safety goggles, gloves, masks) to ensure compliance.
- **Action Recognition & Pose Estimation:** Utilizes advanced pose estimation to track human keypoints and recognize various actions within the lab environment, including:
  - Running
  - Jumping
  - Bending
  - Arm Raising
  - Lying on the Floor (detecting accidents/fainting/slips)
  - Touching Face (critical for preventing contamination in labs)
  - Leaning or Climbing on unstable lab equipment
- **Automated Monitoring:** An automated approach to actively monitor video feeds and log violations, removing the need for manual continuous oversight and ensuring a pristine, compliant pharmaceutical workspace.
